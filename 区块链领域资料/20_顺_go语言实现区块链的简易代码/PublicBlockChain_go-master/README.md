# PublicBlockChain_go 
# 基于goLang的公链实现(具备公链全功能)

# 前言

之前在github上看到一位歪果友人Ivan Kuznetsov用go实现的公链Demo，正好自己也在学习区块链相关知识，于是想记录自己是如何一步步实现基于GoLang的公链项目。喜欢的朋友可以给个star权当鼓励，也可以一起探讨区块链相关知识，我们共同进步。
